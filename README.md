# 🚗 Car & Truck Website

A modern, responsive website showcasing cars and trucks for sale, rent, or display. Built with a clean user interface and dynamic components, this project is ideal for automotive businesses or enthusiasts looking for an online presence.

## 🛠 Features

- 🧭 Sticky navigation bar with smooth scrolling
- 📱 Fully responsive design (mobile, tablet, desktop)
- 📸 Image carousels for featured vehicles
- 🔍 Search and filter capabilities (optional/future)
- 📄 Separate pages for Home, About, Contact, testimonials, and Offerings
